# Divisions

Subdivisions au sein de chaque faction. Chaque joueur appartient à **une division** en plus de sa faction et de son grade. La division détermine ses **rôles in-game accessibles**, son **équipement préféré**, et son **lore RP**.

Un joueur ne peut appartenir qu'à **une seule division** à la fois. Changement possible après 30 jours, validé par un officier.

---

## 🪖 Divisions UNSC

### 1. **7th Marines** — Infanterie de ligne

| Champ | Valeur |
|---|---|
| Rôle | Combat d'infanterie standard, tenue de positions, assaut |
| Loadout préféré | AR + Magnum, BR + SMG, DMR |
| Véhicules | Warthog (passager/conducteur), Mongoose |
| Grade min entrée | Recruit |
| Grade max | Captain |
| Effectif typique en Op | 50–60 % du contingent UNSC |
| Lore | "Helljumpers du sol", troupes les plus polyvalentes du UNSC |

### 2. **ODST** — Orbital Drop Shock Trooper

| Champ | Valeur |
|---|---|
| Rôle | Ops aéroportées, infiltration, frappes rapides |
| Loadout préféré | Silenced SMG, Magnum, DMR, Sniper |
| Véhicules | Falcon (si dispo), Mongoose |
| Grade min entrée | Corporal (transfert depuis Marines) |
| Grade max | Lieutenant |
| Capacités RP | Peut être déployé en avant-garde, peut effectuer des "drops" RP |
| Effectif typique en Op | 2–4 par contingent |
| Lore | Élite parachutiste, "Helljumpers" |

### 3. **SPARTAN-III** — Programme Spartan

| Champ | Valeur |
|---|---|
| Rôle | Force d'élite, missions critiques, leadership tactique |
| Loadout préféré | DMR + Magnum, Sniper, Shotgun |
| Véhicules | Tous |
| Grade min entrée | Sergeant + validation Command Staff |
| Grade max | Captain |
| Capacités RP | Aura d'autorité, accès aux ordres directs Command, peut commander une fire team mixte |
| Effectif typique en Op | 1–2 par contingent (élitisme) |
| Lore | Programme expérimental Beta/Gamma, Reach |

### 4. **Air Cavalry** — Cavalerie aérienne et blindée

| Champ | Valeur |
|---|---|
| Rôle | Pilotage véhicules lourds, support aérien, transport |
| Loadout préféré | Magnum + AR (loadout léger pour pilote) |
| Véhicules | Falcon, Banshee (capturée), Scorpion, Warthog (conducteur) |
| Grade min entrée | Corporal |
| Grade max | Captain |
| Capacités RP | Priorité absolue sur le pilotage des véhicules lourds |
| Effectif typique en Op | 1–3 par contingent |
| Lore | "Hellbringers", pilotes formés sur tous types d'engins |

### 5. **Logistics Corps** — Corps logistique

| Champ | Valeur |
|---|---|
| Rôle | Support, ravitaillement, FOB management, technique |
| Loadout préféré | AR + Magnum, équipement de réparation simulé (lore) |
| Véhicules | Warthog (transport), Mongoose |
| Grade min entrée | Recruit |
| Grade max | Lieutenant |
| Capacités RP | Peut "déployer" des ressources fictives (sacs de munitions, tourelles) en début d'Op via Forge prefabs |
| Effectif typique en Op | 1–2 par contingent |
| Lore | Quartiers-maîtres, ingénieurs de combat |

## 👽 Divisions Covenant

### 1. **Honor Guard** — Garde d'honneur

| Champ | Valeur |
|---|---|
| Rôle | Infanterie lourde, choc frontal, garde des hauts officiers |
| Loadout préféré | Energy Sword, Plasma Rifle, Plasma Launcher |
| Armor Ability | Armor Lock |
| Véhicules | Wraith (Op spéciale) |
| Grade min entrée | Major |
| Grade max | Field Marshal |
| Lore | Élite cérémonielle et de combat |

### 2. **Special Operations** — Spec Ops

| Champ | Valeur |
|---|---|
| Rôle | Infiltration, sabotage, frappes silencieuses |
| Loadout préféré | Plasma Repeater, Needler, Plasma Pistol, Energy Sword |
| Armor Ability | Active Camo |
| Véhicules | Ghost |
| Grade min entrée | Major (transfert depuis Honor Guard ou Ranger) |
| Grade max | Ultra |
| Lore | Élites Spec Ops noirs, frappes éclair |

### 3. **Zealot Order** — Ordre zélote

| Champ | Valeur |
|---|---|
| Rôle | Officiers fanatiques, leadership tactique, autorité religieuse |
| Loadout préféré | Energy Sword, Concussion Rifle, Plasma Pistol |
| Armor Ability | Hologram |
| Véhicules | Banshee |
| Grade min entrée | Ultra + validation Council |
| Grade max | Field Marshal |

| Lore | Caste élite religieuse, équivalent SPARTAN-III |

| Champ | Valeur |
|---|---|
| Rôle | Mobilité haute, assaut aéroporté, reconnaissance |
| Loadout préféré | Focus Rifle, Needle Rifle, Plasma Pistol |
| Armor Ability | Jet Pack |
| Véhicules | Banshee, Ghost |
| Grade min entrée | Major |
| Grade max | Ultra |
| Lore | Sangheili formés au combat zéro-G et atmosphérique |

### 5. **Engineer Corps** — Corps des ingénieurs

| Champ | Valeur |
|---|---|

| Loadout préféré | Plasma Repeater, Plasma Pistol |
| Armor Ability | Drop Shield (équivalent Covenant) |
| Véhicules | Ghost (transport) |
| Grade min entrée | Minor |
| Grade max | Major |
| Capacités RP | Déploiement de barricades / shields RP-simulés via Forge prefabs |
| Lore | Sangheili techniciens (rare in-Reach, alternative narrative) |
