# Système XP — Joueurs vs IA

Document expliquant comment l'XP est gagnée selon la nature de la cible (joueur ou IA) et son rang/type.

---

## Principe central

> **Un joueur réel donne TOUJOURS plus d'XP qu'une IA équivalente.**

Le RP est un jeu de **personnes**, pas de bots. Le combat contre des IA est utile (entraînement, équilibrage Op Firefight), mais il ne doit jamais devenir la voie principale pour grimper en niveau.

---

## Hiérarchie des récompenses (du moins au plus rentable)

### 1. IA basique (Grunt, Marine basique)
- 5–8 XP par kill
- Permet d'apprendre le jeu sans pression

### 2. IA standard (Jackal, ODST, Élite Minor)
- 8–15 XP par kill
- Combat normal en Firefight

### 3. IA élite (Élite Major, Élite Ultra, Hunter, Brute)
- 20–40 XP par kill
- Cibles tactiques en Firefight

### 4. IA boss (Brute Chieftain, Élite Zealot AI)
- 50–80 XP par kill
- Bonus Round Firefight, Op spéciale

### 5. Joueur bas grade (Recruit, Minor)
- 25–30 XP par kill
- Combat PvP standard

### 6. Joueur grade moyen (Corporal, Sergeant, Major, Ultra)
- 45–70 XP par kill
- Cibles plus dures à atteindre

### 7. Joueur grade élevé (Lieutenant, Captain, Spec Ops Officer, Zealot)
- 100–180 XP par kill
- Vétérans expérimentés, gros défis

### 8. Joueur Field Commander / High Council Member
- 250–300+ XP par kill
- High-value target, le Saint Graal

### 9. Joueur SPARTAN-III (UNSC) ou Operation Commander (rôle)
- Modificateur ×1.5 sur la valeur de base
- Ces joueurs sont des "boss" RP

## Plafond IA par session

> Pour empêcher le farm IA exclusif, **30 % maximum de l'XP totale d'une session peut venir de kills IA.** Au-delà, les kills IA suivants comptent à **50 %** de leur valeur.

### Exemple concret
Op Firefight de 25 minutes. Le joueur a accumulé :
- 200 XP de présence
- 100 XP de victoire
- 100 XP d'objectifs
- = 400 XP "non-IA"

→ Plafond IA = 30 % de (XP totale visée). Si on vise 600 XP/session max, plafond IA = 180 XP environ.

Si le joueur a déjà 200 XP de kills IA accumulés, ses kills suivants sont comptés × 0.5 jusqu'à la fin de la session.

**Logique** : encourage les Ops PvP, ne pénalise pas les Op Firefight, mais limite le grind PvE pur.

---

## Multiplicateurs spéciaux

### "First op lifetime" (×1.5)
La toute première Op d'un nouveau joueur : **toute son XP est × 1.5**. Encourage l'onboarding.

### Mentor (×1.0 base + 50 bonus)
Un Sergeant+ qui mentor une recrue lors de sa première Op gagne **+50 XP** post-Op.

### Streak 5 Ops en 14 jours (+200)
Bonus de fidélité.

### MVP vote (+100)
Le joueur le plus voté par ses pairs (vote anonyme post-Op) gagne +100 XP.

### Honorable Duel (+50, Covenant uniquement)
Tuer un adversaire à l'Energy Sword en duel direct (pas de tir de loin avant) → +50 XP bonus côté Covenant. Encourage le RP martial.

## XP par objectifs RP (priorité absolue)

Les objectifs RP donnent **autant ou plus d'XP que les kills**. C'est un choix design : le RP n'est pas un FPS, c'est une simulation militaire.

| Objectif | XP |
|---|---|
| Capture flag (Op Sentinel, Covenant) | 250 |
| Defense flag entire round (UNSC) | 200 |
| Sabotage successful (Covenant) | 150 |
| Silent infiltration 5min (Covenant) | 100 |
| Asset protected critical | 100 |
| Round won | 100 |
| Recon report validated | 50 |
| Operation Commander rôle | +100 |
| Squad/Strike Team Leader rôle | +75 |
| Comportement RP exemplaire | +50 |

→ **Un Squad Leader UNSC qui défend le flag pendant 25 min sans tuer beaucoup peut quand même gagner 400+ XP** par Op (présence + victoire + objectif + rôle + discipline).

## Plafond global de session

| Limite | Valeur |
|---|---|
| Session XP min | 0 |
| Session XP max standard | 600 |
| Exceptions (×1.5 first op, mega-bonus high-value kill) | jusqu'à ~1000 max théorique |

---

## Règles anti-abus

1. **Pas de kill farm sur ami consentant** : si deux joueurs s'arrangent pour se tuer mutuellement (kill swap), sanction = annulation XP de la session
2. **AFK > 50% session** : XP_session = 0
3. **Spam de team kill pour réinitialiser** : sanction immédiate
4. **Camping spawn ennemi prolongé** (>30s sur une zone de respawn) : warning, puis pénalités
5. **Discord vocal d'autre squad utilisé** : XP × 0.5

## Tracking dans la Google Sheet

Les XP par cible sont **logguées** dans l'onglet `MatchResults` :
- Le SL note les kills marquants (kills officier ennemi, kills SPARTAN-III, etc.) pour validation
- Les kills "standards" sont estimés (le scoreboard MCC montre kills/morts, pas les types de cibles)
- En MVP, c'est **estimatif** — on ne peut pas tracker précisément qui a tué qui via MCC
- Le SL utilise son jugement et les retours du squad pour attribuer les bonus

### Format de log par joueur dans MatchResults
```
Player: [SGT HAWK]
Time: 25 min
Kills total (scoreboard): 14
  - estimated breakdown: 10 vs IA Élite, 3 vs joueur Major Cov, 1 vs joueur Spec Ops Officer
  - XP estimé: (10×15) + (3×50) + (1×120) = 420 XP de kills
Objectives: flag defended → +200
Time: 200
Role bonus (SL): +75
Discipline: +50
TOTAL: 945 → CAPPED at 600 (max session)
```
