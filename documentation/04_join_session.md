# Guide Joueur — Comment rejoindre une session

Pour les joueurs participants. Étapes du moment où tu vois l'annonce d'Op au moment où tu joues.

---

## J−24h à J−1h : inscription

### 1. Annonce de l'Op
Tu verras une annonce dans :
- `#unsc-mission-board` (si UNSC)
- `#cov-mission-board` (si Covenant)

Format type :
```
🎯 OP SENTINEL | 2026-05-04 21:00 CET
Host : @cmdr_reed
Map : FOB Alpha
Variante : Sentinel-Defense v1
Briefing : 20:45 dans UNSC-Briefing
Lancement : 21:00
Effectifs : 12 UNSC vs 4 Covenant
Inscriptions : ✅ pour valider
```

### 2. T'inscrire
- Réagis avec **✅** sous le message
- Si tu ne peux plus venir, retire ta réaction au moins **1h avant**
- Dépassement du quota : ordre d'inscription = ordre de priorité

### 3. Confirmer ton install
24h avant l'Op, vérifie que tu as :
- ✅ MCC à jour
- ✅ Game Pack version mentionnée installé (`%USERPROFILE%\AppData\LocalLow\MCC\Temporary\UserContent\HaloReach\`)
- ✅ Steam : ami avec le host
- ✅ Discord : pseudo conforme `[GRADE] Callsign`

---

## J−30min : préparation

### 1. Sois en ligne sur Discord
- Connecte-toi 30 min avant
- Vérifie ton micro/casque
- **Push-to-talk recommandé** pour les canaux d'Op

### 2. Lance MCC en mode AC OFF
- Steam → MCC → Jouer
- Dans le launcher : coche **Anti-Cheat OFF** / **Mod Mode** (selon version)
- Vérifie le bandeau au démarrage : "Anti-Cheat is disabled" doit apparaître
- Va dans **Multiplayer → Custom Games → Halo Reach**
- **Reste dans ce menu** en attendant l'invite Steam

### 3. Vérifie une dernière fois ton équipement
- Casque branché et qui marche
- Volume du micro testé
- Notifications Discord activées

---

## J−15min : briefing

1. **Connecte-toi au canal vocal** :
   - UNSC : `#unsc-briefing-ops`
   - Covenant : `#cov-briefing-ops`
2. Le **Squad Leader / Strike Team Leader** présente :
   - Contexte IC
   - Objectifs primaires et secondaires
   - Ta fire team / strike team
   - Ton **rôle assigné** (Rifleman, Marksman, Sniper, etc.)
   - Ton **loadout**
   - RoE (règles d'engagement)
3. **Pose tes questions maintenant** — pas pendant le match

### 4. Bascule dans ton canal squad
- Le SL te dit dans quel vocal squad aller :
  - `UNSC-Squad-Alpha`, `UNSC-Squad-Bravo`, `UNSC-Squad-Charlie`
  - Ou `COV-Strike-Team-1`, `COV-Strike-Team-2`
- Reste-y pendant tout le match

---

## J−5min : invite Steam

1. Le **host** envoie une invite Steam
2. **Accepte-la** : Steam → notification → Joindre
3. Tu rejoins le lobby Custom Game MCC
4. Vérifie :
   - ✅ Tu es dans la bonne équipe (Defender = UNSC, Attacker = Covenant)
   - ✅ Le pseudo Spartan/Élite affiche bien ton callsign
   - ✅ Le bon loadout / arme primaire est sélectionné

### Si l'invite ne marche pas
- Vérifie que tu es bien ami Steam avec le host
- Vérifie que MCC est bien lancé en AC OFF (l'invite ne marche pas en AC ON pour custom mod games)
- Demande au host de te re-inviter
- Si rien ne marche en 3 min : signal en vocal, le host peut t'ajouter manuellement

---

## J : pendant le match

### Communication
- **Vocal IC strict** : callsigns, formules courtes
- Format type : `[GRADE/CALLSIGN] message`
  - `[CPL HAWK] Mouvement secteur Echo, demande appui`
  - `[SGT REED] Reçu, je redéploie. ETA 30s`
- **Pas de hors-sujet** : si tu as un truc à dire OOC (out-of-character), va sur `#général` après l'Op

### Hiérarchie
- **Tu obéis à ton SL / STL** sans discuter pendant l'Op
- Si désaccord tactique : tu peux suggérer en bref, mais le SL tranche
- Pas de discussion de l'autorité pendant le match

### Comportement
- Pas de team-kill (sanction immédiate)
- Pas de vocal d'autre squad (méta-gaming = XP × 0.5)
- Pas de "GG ez" / langage casual : c'est un Military RP, on incarne
- En cas de bug grave : signal vocal au SL, qui décide

### Mort & respawn
- Tu respawn selon les règles de la variante (8s UNSC, 12s Covenant pour Sentinel)
- À la mort : reste calme, attends le respawn, reprends position
- Pas de "oh putain" (ou équivalent) en vocal pendant que les autres communiquent

---

## Post-match : debriefing

1. Tous reviennent dans le canal vocal de **briefing** (UNSC ou Covenant)
2. **Tour de table 30s/joueur** : ressenti, point notable
3. Le SL fait un **debrief tactique** : 5–10 min
4. Tu peux ensuite retourner où tu veux (général, autres jeux, IRL)

### MVP de l'Op
Vote anonyme via DM Discord au SL → 1 voix par joueur, pas pour soi-même → +100 XP au plus voté.

### Promotions ?
Les promotions sont **traitées chaque lundi soir**, pas immédiatement post-Op. Patience.

---

## Préparation alternative : tu n'as pas le Game Pack

Si tu rejoins à la dernière minute et que tu n'as pas le pack :

1. Demande dans le vocal d'Op si quelqu'un peut te le passer (DM Discord)
2. Ou télécharge depuis `#patch-notes`
3. Installation rapide :
   - Extraire ZIP dans `%USERPROFILE%\AppData\LocalLow\MCC\Temporary\UserContent\HaloReach\`
   - Relancer MCC en AC OFF
4. Si tu ne peux pas dans les temps : tu **abandonnes l'Op**, le host te remplace par un suppléant si possible

→ **Toujours vérifier le Game Pack la veille pour éviter ça.**
